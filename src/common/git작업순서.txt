 [팀장님(깃을 관리하는 사용자)]
 1. github에 저장소 만들기
 2. 작업을 하기 위한 프로젝트 폴더를 이클립스에 만들기 
 3. 로컬저장소 만들기 (폴더 우클릭 > Team > share project > 원하는 폴더 선택 create > finish)
	-> '?' 표시는 아직 git 저장소에 올라가지 않았다는 의미 (untracked 상태)
 4. 스테이지영역에 github 저장소에 올릴 파일 추가하기(add) (파일 선택 후 + 버튼)
 5. 작업디렉토리에 있는 파일을 로컬저장소에 저장하기(commit) (메세지 작성 후 커밋/각각 구별 코드가 존재)
 6. remote 연결하기
 	-> git 원격저장소에 연결하기 위해서 토큰을 발급받기 (1회 - 한 번 발급하면 다시 볼 수 없으므로 잘 관리)
